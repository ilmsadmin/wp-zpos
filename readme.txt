A comprehensive Point of Sale system for WordPress with optional WooCommerce integration, inventory management, warranty tracking, and customer management.

== Description ==

ZPOS is a powerful Point of Sale (POS) plugin for WordPress that transforms your website into a complete sales management system. Whether you're running a retail store, service business, or e-commerce site, ZPOS provides all the tools you need to manage sales, inventory, customers, and warranties.

= Key Features =

* **Complete POS System** - Sell products directly from your WordPress admin
* **WooCommerce Integration** - Optional sync with WooCommerce products, customers, and orders
* **Inventory Management** - Track stock levels with low stock alerts
* **Customer Management** - Comprehensive customer database with purchase history
* **Warranty System** - Manage product warranties with expiration tracking
* **Advanced Reports** - Revenue, sales, and inventory reports with charts
* **Frontend Warranty Check** - Allow customers to check warranty status
* **Responsive Design** - Works perfectly on desktop, tablet, and mobile devices

= WooCommerce Integration =

ZPOS can work independently or integrate seamlessly with WooCommerce:
* Sync products, customers, and orders from WooCommerce
* Maintain separate database for POS-specific data
* Two-way synchronization options
* Works with or without WooCommerce installed

= Getting Started =

1. Install and activate the plugin
2. Run the setup wizard to configure basic settings
3. Choose whether to sync with WooCommerce
4. Start selling with the built-in POS system
5. Use `[zpos_warranty_check]` shortcode for frontend warranty checking

= Requirements =

* WordPress 6.0 or higher
* PHP 7.4 or higher
* WooCommerce 8.0+ (optional)

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/zpos` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the ZPOS->Setup Wizard screen to configure the plugin
4. Start using the POS system from the ZPOS menu in your WordPress admin

== Frequently Asked Questions ==

= Do I need WooCommerce to use ZPOS? =

No, ZPOS works independently. WooCommerce integration is optional and can be enabled during setup.

= Can I use ZPOS on mobile devices? =

Yes, ZPOS is fully responsive and optimized for tablets and mobile devices.

= How do customers check their warranty? =

Use the `[zpos_warranty_check]` shortcode on any page or post. Customers can search by phone number or serial number.

= Can I import my existing WooCommerce data? =

Yes, ZPOS can sync existing products, customers, and orders from WooCommerce during setup.
